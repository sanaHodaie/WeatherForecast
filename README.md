## 📖 معرفی | Introduction

**پیش‌بینی آب و هوا** یک اپلیکیشن وب مدرن است که به شما امکان می‌دهد وضعیت جوی شهرهای مختلف را مشاهده کنید. این برنامه با طراحی زیبا، ریسپانسیو و امکانات پیشرفته ساخته شده است.

**Weather Forecast** is a modern web application that allows you to view the weather conditions of different cities. Built with beautiful design, responsive layout, and advanced features.
