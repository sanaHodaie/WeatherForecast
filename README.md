---

## ✨ Features

| Icon | Feature | Description |
|------|---------|-------------|
| 🔍 | **Advanced Search** | Search cities in both Persian and English |
| ❤️ | **Favorites** | Save and manage favorite cities in browser storage |
| 🌙 | **Dark Mode** | Toggle between light and dark themes with auto-save |
| 🌐 | **Multilingual** | Full support for Persian and English languages |
| 📄 | **PDF Report** | Export weather information as A4 PDF document |
| 📊 | **Temperature Chart** | Graphical display of 7-day temperature forecast |
| 📱 | **Responsive Design** | Fully compatible with mobile, tablet, and laptop |
| ✨ | **Beautiful Animations** | Glow effects matching weather conditions |
| ☀️ | **3D Elements** | Animated sun, clouds, and birds in background |
